Splay Tree (Using BST)
====================================

Project Description
-------------------
It's a splay tree.

Installation
------------
At the moment, just do it if you want to write tests.

.. code-block:: bash

   pip install -r requirements.txt

Usage
-----
Splay tree class is ready to use. Usage could be found in tests/test_splay_tree_map.py.
.. code-block:: bash

    pytest
